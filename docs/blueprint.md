# **App Name**: TrackWise

## Core Features:

- Income and Expense Tracking: Allows users to log their income and expenses with date, amount, and category.
- Categorization: Categorizes transactions (e.g., Food, Travel, Bills) for clear financial understanding.
- Reporting and Analytics: Generates weekly/monthly reports and analytics on income and expenses.
- Budgeting: Sets monthly budgets and provides alerts when budgets are exceeded.
- Data Visualization: Visualizes financial data using charts and graphs to identify trends and patterns.
- Financial Insights Tool: Provides AI-driven recommendations for savings based on user spending patterns by considering various sources of information related to financial wellness.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) for a calm, trustworthy feel.
- Background color: Light grey (#F0F4F8), nearly white, to provide a neutral, uncluttered backdrop that enhances the focus on content and data presentation.
- Accent color: Coral (#FF7F50), offering a lively and contrasting highlight to guide user interaction and add warmth to the interface.
- Body and headline font: 'Inter', a sans-serif font, is used for both body text and headlines, lending a clean and modern look.
- Use simple, outline-style icons to represent different transaction categories and actions.
- Maintain a clean, minimalist layout with clear sections for income, expenses, reports, and budgets.
- Use subtle transitions and animations to provide feedback and guide users through the app.
